# Autopilot

In Out-of-Box Experience, all of the requirements to run `Get-WindowsAutopilotInfo` can be handled in a single command line after pressing `Shift + F10` to open a command prompt

{% code title="Command Prompt" %}
```
powershell iex(irm autopilot.ps1.osdeploy.com)
```
{% endcode %}
